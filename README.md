# GameProg

# 全体

swingで構成,mainにループがあってそこでアニメーション関係を制御してたりする。

# 構成

こんな感じ

* contentPane
    * title
    タイトル画面.
    * main
    メイン画面.
        * menu
        メニュー画面.
            * confs
            現在の状況を確認.
            * unorg
            ユニットを管理.
            * tact
            ユニットの戦術を研究.
            * sortie
            ステージに挑戦.
        * war
        ステージで戦闘

# 仕様

敵と戦闘を行って最後までユニットが残っていたチームが勝ち.
